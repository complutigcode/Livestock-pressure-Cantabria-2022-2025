📖 Overview

This repository contains a high-resolution GPS tracking dataset of bovine livestock across the diverse ecosystems of Cantabria, Spain. The data spans four consecutive years (2022-2025), providing a unique longitudinal perspective on grazing behavior, land-use intensity, and seasonal migration (transhumance) between coastal valleys and high-mountain pastures.

The primary goal of this project is to quantify Livestock Pressure—the environmental impact and spatial distribution of cattle—to support sustainable range management and ecological research.
